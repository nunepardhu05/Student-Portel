# 🎓 Student Portal - Single Page Web Application

## 📌 Overview

The **Student Portal** is a **Single Page Web Application (SPA)** developed using **HTML, CSS, and JavaScript**. It allows users to manage student information, courses, and attendance through a simple and interactive interface without reloading the webpage.

All sections are displayed dynamically on a **single page**, providing a smooth and user-friendly experience.

---

## ✨ Features

- 📋 Add and manage student details
- 📚 View available courses
- ✅ Track student attendance
- 🔄 Navigate between sections without page refresh
- 🎨 Responsive and simple user interface
- ⚡ Fast and interactive single-page design

---

## 🛠 Technologies Used

| Technology | Purpose |
|-------------|---------|
| HTML5 | Structure of the webpage |
| CSS3 | Styling and layout |
| JavaScript | Dynamic content and page switching |

## ⚙ Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/student-portal.git
```

### 2. Move to Project Directory

```bash
cd student-portal
```

### 3. Run the Project

Open the `index.html` file directly in your browser.

or

Use VS Code Live Server:

1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.
